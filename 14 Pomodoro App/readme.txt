The Pomodoro Timer Application is a time management tool based on the Pomodoro Technique, which was developed by Francesco Cirillo in the late 1980s
It has Work Interval of 25 minutes, Break Interval of 5 minutes and Long Break Interval of 20 minutes (after 4 work intervals).

About the Application
In the tomato picture, there is a timer with two buttons, one for starting and one for resetting the timer. The top of the screen has a text label that changes from work to break depending on the interval.
