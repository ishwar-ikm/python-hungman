This is a quiz game in which there are 12 quetions in true/false type.
Depending on whether the answer is true or false, the player receives one point for each correct answer and zero for each incorrect response.
At the end of the quiz player can see their final score 

The questions are taken from https://opentdb.com/ 
Since every thing is in object oriented structure we can simply change the data.py for new types of questions without changing other classes
