This is a famous snake game in which the snake must obtain its food and grows as it eats. The game gets over as soon as the snake hits the wall or the body of the snake.

food.py: Contains the food class which defines the properties of food and its position. Has a method which randomly sets the position of the food.

scoreboards.py: Contains the score class that keeps track of the score scored by the user and updates it accordingly. This file also defines the game over screen text.

snake.py: Contains the snake class which defines the body of the snake and its behaviour. It define the initial poisition, contains method to extend the body of the snake and move the snake according to the user input.

main.py: This file contains all the functionality of the game. It contains object of all the classes and the logic for the game to work.
