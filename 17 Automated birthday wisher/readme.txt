This is a Python program that automates the process of sending birthday wishes via email. It reads birthday data from a CSV file and sends personalized emails to the recipients. The program uses the smtplib library to establish an SMTP connection and send emails.

Features:
Automatically sends personalized birthday wishes via email
Reads birthday data from a CSV file
Uses customizable email templates
Supports Gmail as the email service provider

User can personalize the message in letter_templates and add or edit birthdays in birthday.csv file.
